> Add your XboxHDKey to "xbhddkey.ini"

```

Title=Splinter Cell: Double Agent
SigKey=61676F08B31D71DCF8E02903CB5164BB
AuthKey=0E07D758C668860E11BD19181569B838
DataFile=*.sav
```
